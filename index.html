About

Solo UK indie game developer and software engineer. Here you will find my projects, achievements, and acknowledgements.

Born and raised in England, graduated Bournemouth University with a First Class honours in BSc Games Software Engineering. I've always enjoyed coding, game development is my favorite outlet for it.

Projects

Game: What Goes Around Comes Around

Created for a 48 hour University Game Jam with the theme "Loop". I worked in a team of two as a co-developer, we created a First Person Shooter where projectiles orbit around a small planet that the player must avoid. Each wave lets the player choose an upgrade to combat the increasing number of enemies.

This project was made in Unity using C#. I implemented the projectiles, planet-based gravity, minimap, and wave system. We won 3rd place overall which I'm very proud of!

Game: The Wild Frontier

Created for a 48 hour University Game Jam with the theme "Centenary". I worked in a team of four as one of the lead developers, we created a city building game set in the wild west where you must maintain your town for a hundred in-game years. I implemented the grid-placement, building selection, camera movement, and post-processing.

This project was made in Unity using C#. Working in a team with distinguished roles really let this project come together. It was a fun experience delegating tasks and connecting other team members work together!

Game: INHABITED

Created for a 48 hour GameMaker game jam with the theme "Small World". I decided to create a small horror game where you play as a survivor of a ship crash. To win you must collect all the power cells that are scattered on this small planet whilst evading the monster by shining your flashlight at it. I created everything; code, sound effects, story, and artwork.

Created back in 2020, I used GameMaker Language (which has a C-style syntax) for the logic, and OpenGL to create the shadow masking shader. I achieved 11th place out of 69. Fun fact, all sound effects were made with my voice!
