Raycasting Engine Test
Here is my raycasting demo! This was actually the first thing I pushed to the GitHub repo, I made it before creating the site. It took me several attempts to get it right (as you can tell if you view the commit logs). The map that's loaded never changes but the colours do if you reload the page. If you click on it, you can move around with the Arrow Keys.

It's powered by JavaScript and is drawn to the canvas element via stroke function calls.
